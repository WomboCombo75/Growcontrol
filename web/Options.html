MJPEG streams

Optional mjpg-streamer

You can leave Stream URL empty: Start MJPEG server fills it automatically. Override by typing a URL. Use your Pi’s LAN host when you use the Dashboard from another device.

Assign sensors (Ctrl/Cmd-click); leave none selected → show on all plants. Click Save webcam streams after edits. Set the install path and camera options below, then Save MJPEG settings.

Built-in camera (mjpg-streamer only)

Resolution and FPS need Stop → Start. Exposure, gain, and image tuning below apply live while the stream runs (about half a second after you change a value).

JPEG quality mainly affects software re-encoding; many USB cameras output MJPEG directly — exposure and gain matter most in low light.

Streams show in Dashboard → Analytics for mapped sensors.
