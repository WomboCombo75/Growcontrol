Webcam (MJPEG)

Optional mjpg-streamer

MJPEG URL: use your Pi’s LAN IP (e.g. http://192.168.x.x:8080/?action=stream), not localhost, so phones/PCs can open the stream.

Assign sensors (Ctrl/Cmd-click); leave none selected → show on all plants. Set install path, then Start or Stop MJPEG server (HTTP port from the URL, default 8080). More: mjpg_streamer in settings.json or GROWCONTROL_MJPG_STREAMER_ROOT.

Streams show in Dashboard → Analytics for mapped sensors.
